# MINE_BOT_OS

## Обзор
Проект представляет собой Telegram-бота для автоматизации контента в каналах о Minecraft-модах, интегрированного с Flask-веб-панелью.

## Основные функции
- **ИИ-генерация:** Автоматическое создание постов для Telegram на основе ссылок или описаний с использованием Groq (Llama 3).
- **Очередь постов:** Планирование публикаций через APScheduler.
- **Работа с изображениями:** Автоматическое наложение водяных знаков.
- **Комментарии:** Автоматический анализ и ответы на комментарии в группах.
- **Веб-панель:** Flask-интерфейс для управления очередью, статистикой и настройками.

## Установка
1. Установите Python 3.10+.
2. Создайте файл `.env` в корневой папке и добавьте необходимые ключи:
   - `BOT_TOKEN`: Токен вашего бота.
   - `GROQ_API_KEY`: API ключ Groq.
   - `ADMIN_IDS`: ID администраторов через запятую.
3. Установите зависимости:
   `pip install -r requirements.txt`
4. Запустите:
   `python main.py`
