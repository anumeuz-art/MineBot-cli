# 📦 MINE_BOT_OS v2.8 — AI Mod Manager for Telegram

**MINE_BOT_OS** — это комплексная экосистема для автоматизации Telegram-каналов по тематике Minecraft. Проект объединяет мощь современных языковых моделей (ИИ), веб-технологий и автоматизированной обработки медиафайлов.

---

## 🚀 Основные функции

### 1. 🤖 Интеллектуальная генерация контента (AI Engine)
Бот использует модель **Llama 3.3 70B** через API Groq для создания постов.
*   **Генерация по ссылке:** Вы отправляете ссылку на мод (CurseForge, Modrinth), бот сканирует страницу, извлекает описание и создает структурированный пост.
*   **Стили переписывания (Rewrite):** 
    *   *Short* — кратко и по делу.
    *   *Detailed* — подробный обзор функций.
    *   *Funny* — молодежный сленг и много эмодзи.
    *   *Pro* — профессиональный обзор для опытных игроков.
*   **Мультиязычность:** Автоматический перевод контента на узбекский (UZ), русский (RU) и английский (EN).
*   **Умные хештеги:** Авто-генерация до 5 релевантных тегов (всегда начинается с `#Minecraft`).

### 2. 🌐 Веб-панель управления (Dashboard)
Полноценный интерфейс на **Flask** для управления проектом без использования Telegram.
*   **Менеджер очереди:** Просмотр запланированных постов, изменение их порядка (Drag-and-Drop), редактирование текста и времени.
*   **Аналитика карточками:** Отдельная статистика для каждого канала (Подписчики, Рост за 24ч, 7 дней, 30 дней).
*   **Настройки Smart Queue:** Установка интервала автоматической публикации и текста подписи.
*   **Prompt Engine:** Редактирование системного промпта ИИ прямо из браузера с поддержкой HTML-тегов.

### 3. 🖼 Обработка медиа и водяные знаки
*   **Watermark Manager:** Поддержка загрузки до 5 различных логотипов. Выбор активного знака через веб-панель.
*   **Авто-наложение:** Бот автоматически накладывает выбранный логотип на все фото и альбомы (до 10 фото в одном посте).
*   **Проксирование медиа:** Фотографии из Telegram отображаются в веб-панели через безопасный прокси-сервер.

### 4. 📅 Автоматизация и планирование
*   **Smart Queue:** Автоматическая постановка постов в очередь с заданным интервалом (например, каждые 6 часов).
*   **Exact Time:** Возможность запланировать пост на конкретную дату и минуту.
*   **Авто-опросы:** Каждые 3 дня бот генерирует и публикует интерактивный опрос для поднятия активности в канале.

## 🛠 Техническая архитектура и Логика

### Структура модулей:
*   **`main.py`**: Ядро системы. Запускает Telegram-бота, Flask-сервер в отдельном потоке и планировщик задач APScheduler.
*   **`handlers.py`**: Обработка всех команд в Telegram. Реализует логику "Черновика" (Draft), где пользователь может редактировать пост перед публикацией.
*   **`ai_generator.py`**: Логика взаимодействия с ИИ. Очистка текста от лишних ссылок, фильтрация хештегов и форматирование под стандарты Telegram (HTML).
*   **`database.py`**: SQLite3 база данных. Хранит очередь постов, историю роста подписчиков, настройки и активные промпты.
*   **`publisher.py`**: Отвечает за физическую отправку сообщений в Telegram. Обрабатывает ошибки и добавляет рекламные подписи.
*   **`watermarker.py`**: Использует библиотеку `Pillow` для динамического масштабирования и наложения логотипа на изображения разных разрешений.
*   **`webapp.py`**: Backend веб-панели. Реализует API для фронтенда и управляет сессиями.

### Логика публикации:
1.  **Прием данных:** Пользователь отправляет ссылку или фото.
2.  **Обработка ИИ:** Генератор создает текст, учитывая выбранный стиль и язык.
3.  **Водяной знак:** Если есть фото, на него накладывается активный логотип из базы данных.
4.  **Выбор канала:** Бот запрашивает, в какой из подключенных каналов отправить контент.
5.  **Очередь:** Пост либо уходит сразу, либо попадает в таблицу `pending_posts` с расчетом времени (текущее макс. время в очереди + интервал).
